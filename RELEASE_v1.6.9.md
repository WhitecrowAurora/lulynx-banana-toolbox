# v1.6.9

本次更新重点修复了图生图兼容性问题，并补上了多 API 站点切换能力，日常使用会更稳、更方便。

## 新增

- 新增 `API 站点列表`，现在可以在设置中保存多个 API 站点配置，并在不同站点之间快速切换。
- 保存的站点配置会同时记录 `API 端点 / API Key / New-API-User / Provider / Model`。
- 备份与恢复现在会包含已保存的 API 站点列表，换机或重装后恢复更完整。

## 改进

- 优化了图生图请求兼容性。
- 对支持官方图片编辑接口的站点，客户端会优先走 `/images/edits` 的 multipart 请求。
- 对旧式兼容站点，仍保留原有的兼容回退逻辑，减少部分站点出现 `429`、`524`、无生成记录等问题的概率。
- 优化余额查询兼容性，支持部分站点通过 `New-API-User` + `/api/user/self` 查询额度。

## 修复

- 修复震动反馈开关不生效的问题，现在设置项会正确同步到实际交互反馈。
- 修复部分页面仍绕过震动设置直接触发系统触觉反馈的问题。
- 修复设置页版本号与自动更新检测版本未同步的问题，当前版本更新为 `v1.6.9`。
- 清理并修复了一批编译与分析问题，当前版本已通过分析检查并完成 release 构建验证。

## 使用说明

- 如果你有多个中转站或不同上游，进入 `设置 -> API 配置` 即可保存当前站点，并随时切换。
- 如果某个站点余额查询需要 `New-API-User`，可以在设置中直接填写。
- 如果之前图生图经常报错，这个版本建议优先更新。

## 其他

- 本次 release 已完成 Android `release APK` 构建验证。
